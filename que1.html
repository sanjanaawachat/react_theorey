que1: what is Higher order components? ans: 1] Higher-order components or HOC is the advanced method of reusing the component functionality logic. It simply takes the original component and returns the enhanced component. syntax: const EnhancedComponent = higherOrderComponent(OriginalComponent); 2] Reason to use Higher-Order component: 1)Easy to handle 2)Get rid of copying the same logic in every component 3)Makes code more readable
